# Password-Strength-Checker
Password Strength Checker is a cyber security tool built using Python that helps users create secure passwords by checking password complexity and strength.
# Project purpose 
The purpose of this project is to help users create strong and secure passwords. Weak passwords are easy for hackers to guess, which can lead to unauthorized access and data breaches. This Password Strength Checker analyzes a password and informs the user whether it is Weak, Medium, or Strong, helping improve cybersecurity awareness and account security.
